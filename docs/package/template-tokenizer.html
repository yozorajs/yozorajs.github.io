@yozora/template-tokenizer


Templates for creating a Yozora tokenizer (InlineTokenizer / BlockTokenizer).

Install​

  • npm
  • Yarn
npm install -g @yozora/template-tokenizer
yarn add -g @yozora/template-tokenizer

Usage​

$ yozora-template-tokenizer
