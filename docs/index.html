Introduction

🎉 What is "yozora" ?#

The name yozora is the Roman sound of Japanese よぞら, taken from the lyrics of the song "花鳥風月" by the band Sekai no Owari.

Yozora is a monorepo that contains a pluggable markdown parser kernel @yozora/core-parser and several implemented tokenizers such as @yozora/tokenizer-autolink for resolving the specific tokens.

✨ Features#

  • Robust, all codes written in typescript with strictly static type checking and well tested with a continuously rich test cases.

  • Tidy, no third-party dependencies (except for tiny-invariant, which also may be removed in the future).

  • Fast and efficient, timing complexity is the number of tokenizers multiplied by the total string length processed. And supports streaming input through iterators.

  • Compatibility, the parsed syntax tree is compatible with the one defined in [Mdast][mdast-homepage].

  • Fully support the GFM specification and passed all of the 600+ examples mentioned in the specification (except https://github.github.com/gfm/#example-653).

    See @yozora/parser-gfm or @yozora/parser-gfm-ex for details.

  • Implement additional grammars such as Admonitions, Footnotes, Formulas, etc., and are built into the parser @yozora/parser.

📝 Usage#

  • YozoraParser
  • GfmParser
  • GfmExParser

@yozora/parser is a markdown parser with rich built-in tokenizers.

import YozoraParser from '@yozora/parser'
const parser = new YozoraParser({ shouldReservePosition: true })parser.parse('source content')

@yozora/parser-gfm is a markdown parser that implements all (without extensions) the specifications mentioned in the Github Flavor Markdown Spec.

import GfmParser from '@yozora/parser-gfm'
const parser = new GfmParser({ shouldReservePosition: true })parser.parse('github flavor markdown contents')

@yozora/parser-gfm-ex is a markdown parser that implements all (including extensions) the specifications mentioned in the Github Flavor Markdown Spec.

import GfmExParser from '@yozora/parser-gfm-ex'
const parser = new GfmExParser({ shouldReservePosition: true })parser.parse('github flavor markdown contents (with gfm extensions enabled)')
